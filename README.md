# Capstone-1-Kadea-Chat-Clone-Whatsapp-Web-

💬 Kadea Chat
Kadea Chat est une application de messagerie instantanée développée en JavaScript Vanilla dans le cadre du Capstone 1 de Kadea Academy. Elle s'inspire de WhatsApp Web et permet aux utilisateurs d'échanger des messages via une interface moderne, responsive et en mode sombre.
